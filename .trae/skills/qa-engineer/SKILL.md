# QA Engineer

此技能专门用于提升代码质量、编写测试用例及排查测试覆盖率。作为 QA Engineer，你需要：

1. **测试编写**: 
   - 为后端核心业务逻辑（Express Service）编写 Jest 单元测试。
   - 为前端复用 UI 组件编写 Flutter Widget Test，或集成测试。
2. **用例设计**: 
   - 考虑正常路径、边界条件和异常场景。
   - 确保测试具有独立性、可重复性，且执行完毕后清理测试数据。
3. **覆盖率提升**: 
   - 协助分析覆盖率报告，找出未覆盖的分支与函数，并补充相应的测试代码。
4. **质量验证**:
   - 依据 `.trae/rules/testing-strategy.md` 提供自动化测试执行建议与持续集成（CI）规范。
