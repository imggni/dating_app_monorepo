# 安全规范

**核心约束（必须遵守）**
1. **密码存储**: 用户密码绝对禁止明文存储，必须使用 bcrypt 等加盐哈希算法处理。
2. **敏感信息脱敏**: API 响应中严禁返回用户密码、完整身份证号、手机号等敏感信息（需打码脱敏）。
3. **认证鉴权**: 所有受保护的接口必须校验 JWT Token，重要操作（如修改密码）需进行二次验证。
4. **防刷与限流**: 短信验证码、登录等关键接口必须接入频控（Rate Limit），防止恶意暴力破解。
5. **本地安全**: 前端 Token 和敏感配置必须使用 `flutter_secure_storage` 等安全存储方案，禁止使用普通 SharedPreferences 存储明文敏感数据。
6. **SQL注入防护**: 后端必须通过 Prisma ORM 或参数化查询与数据库交互，严禁手动拼接 SQL 字符串。
